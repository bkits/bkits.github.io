Docker Cheatsheet

Install (easy but not secure)

curl -fsSL https://get.docker.com -o get-docker.sh

sudo sh get-docker.sh

sudo docker version

Test

sudo docker run docker/whalesay cowsay HELLO

Commands

commands will require sudo if the user is not in the docker group

run - runs image or downloads and runs image (docker run <image>), -d command run in dettach mode returns id (use this if you want to use this terminal)

attach - use this to attach terminal to container if docker run -d <image> was used (docker attach <id>, you can use first few characters if they are unique from other ids)

ps - list of running containers (docker ps), -a option list even the ones not running (ps -a). (NOTE: containers not images!!!)

stop - stops running container (docker stop <name>, where name is the field from docker ps)

rm - remove perminatly container (docker rm <name>)

images - list of images (docker images)

rmi - remove image (docker rmi <repository>, where repository is the field from docker images)

pull - only download (not run) image (docker pull <image>)

exec - execute a command in a container (docker exec <name> <command>)

Examples

docker run docker/whalesay cowsay HELLO

docker run -d --name=whale docker/whalesay sleep 100

docker ps

docker stop whale

docker start whale

docker exec whale cowsay hello

docker ps -a
